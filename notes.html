<html>
Defines the start and end of a HTML document
<head>
Contains information about the document, such as the title, meta data, and styles
<body>
Contains the content of the web page that is visible to the user
<h1>
Defines a heading on a web page, with <h1> being the most important heading and <h6> being the least important
<p>
Defines a paragraph of text on a web page
